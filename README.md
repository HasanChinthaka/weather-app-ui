# Setup env file

1. Rename exmple.env --> .env

2. Register on [OpenWeatherMap](https://openweathermap.org/) to obtain an API key.

3. Add open wather API Key

# Run Project

1. Install dependencies

   ```bash
   npm install
   ```

2. Recheck .env file variable

3. Run project local

    ```bash
      npm run dev
    ```
